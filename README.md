## What is it?
> This is a platform to people discover and help NGO's (Non-governmental organizations).

## :computer: Technologies:
* NodeJS
* SQL
* React
* React Native
## :interrobang: Instructions:

### Installing the backend modules

```sh
$ cd backend
$ npm install
```

### Running the backend

```sh
$ cd backend
$ npm start
```
